# Calculator
Project Title: GUI-based Calculator in Java

Description:
Developed a fully functional GUI-based Calculator using Java Swing and AWT, capable of performing basic arithmetic operations such as addition, subtraction, multiplication, division, and percentage calculations.
The application features a responsive and intuitive graphical user interface (GUI) with buttons for digits, operations, and additional functionalities like clearing input and backspacing.

Technologies Used:

Java (Core Java, OOPs concepts)
Swing & AWT (GUI components like JFrame, JButton, JTextField)
Key Features:
✔ User-friendly interface with a structured layout
✔ Supports basic arithmetic operations (+, -, *, /, %)
✔ Real-time input display and backspace functionality
✔ Implemented using Event Handling (ActionListener) for button interactions
✔ Proper memory management for handling calculations efficiently

Role & Responsibilities:

Designed and implemented the GUI layout using Java Swing components.
Managed event-driven programming to handle user inputs effectively.
Implemented logic for performing calculations and displaying results.
Ensured smooth functionality with proper exception handling.
